# Changelog

All notable changes to the Lara-Veil project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2024-01-17

### Added
- Complete hook system with action/filter support
- Advanced MediaForgeService with image processing capabilities
- Plugin and theme lifecycle management
- Service provider auto-discovery
- Permission-based security policies
- Built-in update system with rollback support
- RESTful API endpoints for plugins, themes, and media
- Artisan console commands for management
- Database migrations for plugin, theme, and media tables
- Configuration files for customization
- Comprehensive logging and monitoring

### Changed
- Restructured as a Composer package

### Fixed
- Media file handling and cleanup
- Hook execution priority
- Service provider registration

## [1.0.0] - 2024-01-01

### Added
- Initial release
- Core extensibility framework
- Basic plugin and theme support
- Hook system foundations
